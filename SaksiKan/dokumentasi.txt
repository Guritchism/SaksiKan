DOKUMENTASI SISTEM SAKSIKAN - SISTEM PEMESANAN TIKET PERTUNJUKAN

A. FITUR UTAMA

1. Manajemen Pengguna
   - Registrasi akun baru
   - Login/logout pengguna
   - Sistem peran (admin dan pengguna regular)
   - Manajemen profil pengguna

2. Manajemen Pertunjukan (Admin)
   - Menambah pertunjukan baru
   - Mengedit informasi pertunjukan
   - Menghapus pertunjukan
   - Unggah poster pertunjukan
   - Pengaturan lokasi dan detail pertunjukan

3. Manajemen Jadwal (Admin)
   - Menambah jadwal untuk pertunjukan
   - Mengatur jumlah kursi tersedia
   - Mengatur harga tiket
   - Mengedit/menghapus jadwal
   - Status aktif/nonaktif jadwal

4. Sistem Pemesanan
   - Pemilihan pertunjukan dan jadwal
   - Pemesanan tiket dengan jumlah yang diinginkan
   - Kode booking unik
   - Status pembayaran
   - Riwayat pemesanan

5. Dashboard Admin
   - Manajemen pengguna
   - Ikhtisar pertunjukan
   - Pemantauan pemesanan
   - Verifikasi pembayaran

B. IMPLEMENTASI TEKNIS

1. Kerangka Kerja Backend
   - Flask (kerangka kerja web Python)
   - SQLAlchemy untuk database ORM
   - Flask-Login untuk manajemen autentikasi
   - Werkzeug untuk keamanan kata sandi

2. Basis Data
   - Basis data SQLite
   - Model:
     * User (pengguna)
     * Show (pertunjukan)
     * Schedule (jadwal)
     * Booking (pemesanan)

3. Frontend
   - Template HTML dengan Jinja2
   - Bootstrap untuk tampilan
   - JavaScript untuk interaktivitas
   - Penanganan formulir dengan WTForms

4. Keamanan
   - Penyandian kata sandi
   - Proteksi CSRF
   - Kontrol akses berbasis peran
   - Unggah file yang aman

5. Fitur Teknis Tambahan
   - Penyimpanan sementara untuk optimasi kinerja
   - Sistem pencatatan
   - Penanganan kesalahan
   - Manajemen unggah file
   - Pembatasan ukuran file

C. STRUKTUR APLIKASI

1. Rute Utama
   - / (halaman utama)
   - /login
   - /register
   - /show/<id>
   - /book/<schedule_id>
   - /admin/*
   - /bookings
   - /verify/<code>

2. Direktori
   - /templates: File HTML
   - /static: Aset statis (CSS, JS, gambar)
   - /logs: File log aplikasi
   - /instance: Basis data dan file konfigurasi

D. PANDUAN PENGGUNAAN

1. Untuk Pengguna
   - Registrasi akun baru
   - Jelajahi pertunjukan yang tersedia
   - Pilih jadwal dan pesan tiket
   - Cek status pemesanan
   - Lihat riwayat pemesanan

2. Untuk Admin
   - Akses dashboard admin
   - Kelola pertunjukan dan jadwal
   - Pantau pemesanan
   - Kelola pengguna
   - Verifikasi pembayaran

E. KEBUTUHAN SISTEM

1. Dependensi
   - Python 3.x
   - Flask dan ekstensinya
   - SQLite
   - Peramban web modern

2. Konfigurasi
   - SECRET_KEY untuk keamanan
   - Konfigurasi folder unggah
   - URI basis data
   - Pengaturan penyimpanan sementara
